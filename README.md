# Clean Distro

A minimal, aesthetically clean Linux distribution built on Arch Linux with Hyprland.

This distro provides a beautiful foundation with modern tools and theming, allowing you to choose your own applications and workflow.

## Features

- **Hyprland** - Modern Wayland compositor with beautiful animations
- **Ghostty** - Fast, GPU-accelerated terminal emulator
- **4 Curated Themes** - Tokyo Night, Catppuccin, Rose Pine, and Gruvbox
- **Minimal Core** - Only essential tools, you choose the rest
- **Web App Framework** - Turn any website into a native-like application
- **Hardware Support** - Conditional installation for NVIDIA, Intel, AMD, and Apple hardware

## Installation

```bash
curl -fsSL https://raw.githubusercontent.com/YOUR_USERNAME/YOUR_REPO/main/boot.sh | bash
```
